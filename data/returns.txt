Return Policy:
- Returns accepted within 30 days of delivery for most items
- Product must be in original condition with all packaging and accessories
- Return shipping is free for defective items or shipping errors
- For non-defective returns, customer pays return shipping unless noted otherwise
- Refunds are processed within 5-7 business days after we receive the returned item
- Original shipping costs are non-refundable unless the return is due to our error
- Gift returns will be issued as store credit to the recipient

Return Exceptions:
- Electronics must be returned within 14 days
- Undergarments and personal care items cannot be returned for hygiene reasons
- Custom or personalized items cannot be returned unless defective
- Discounted items marked as "Final Sale" cannot be returned

Return Process:
1. Log into your account and select the order with items you wish to return
2. Select the specific items and reason for return
3. Print the return label (free for defective items, $7.99 deducted from refund for other returns)
4. Package the items securely with all original packaging and accessories
5. Attach the return label to the outside of the package
6. Drop off at any authorized shipping location
